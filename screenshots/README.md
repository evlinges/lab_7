# Скріншоти для звіту

Помістіть ваші скріншоти в цю папку з такими назвами:

1. `seed-database.png` - Скріншот терміналу з заповненням БД
2. `main-page.png` - Скріншот головної сторінки
3. `create-post.png` - Скріншот форми створення поста
4. `analytics.png` - Скріншот аналітичної панелі

Або використовуйте будь-які інші назви, але оновіть посилання в `REPORT.md`.
